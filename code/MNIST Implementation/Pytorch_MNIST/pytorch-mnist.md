# 文件目录结构

大致结构如下：
```
├─Pytorch_MNIST
│  ├─checkpoint
│  │  ├─model-mnist.pth 保存的模型
│  ├─data              MNIST数据集（文件过大未上传)
│  ├─mnist.ipynb	   Jupyter Notebook版文件
│  ├─mnist_cnn.py		  py版文件	
│  ├─Digraph.gv.pdf		 CNN网络结构图
│  ├─README.md		 文件结构说明的markdown文件	
│  ├─运行结果.jpg		 py文件运行截图
```
